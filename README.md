# EYE Q&A Game Show

Hosted edition of the Express Yourself! Expo Q&A Game Show developed by InTogether.

## Publish with GitHub Pages

Set Pages to deploy from the `main` branch and the repository root. The entry page is `index.html`.

## Updating

Replace changed files in the local GitHub Desktop repository, commit, and push. GitHub Pages republishes automatically.

## Offline version

Keep the original standalone HTML/ZIP as the offline backup.
